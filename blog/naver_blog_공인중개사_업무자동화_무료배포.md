## 공인중개사 업무 자동화, 이제 크롬 확장으로 해결하세요

공인중개사님, 이런 상황 익숙하시죠?

- 매물 등록할 때마다 경쟁 매물 광고 문구를 하나씩 열어 직접 복사

이 문제를 해결하기 위해, 현직 공인중개사의 실제 피드백을 바탕으로 개발한 **무료 크롬 확장 프로그램 2종**을 공개합니다.

## 도구 1. 네부스캐너 — 경쟁 매물 광고 문구 자동 수집 + AI 문구 생성

### 이런 분께 필요합니다

네이버 부동산에 매물을 등록할 때, 가격이나 면적 같은 기본 정보는 경쟁 매물과 다를 게 없습니다. 손님이 어떤 중개사에게 연락할지 결정하는 건 결국 **한줄 특징과 상세 설명**입니다.

그런데 지금 이 광고 문구를 어떻게 작성하고 계신가요? 대부분 경쟁 매물을 하나씩 열어보면서 좋은 문구를 수동으로 복사하고 있을 겁니다. 이 작업이 은근히 시간을 잡아먹습니다.

### 네부스캐너가 하는 일

**네부스캐너**는 네이버 부동산 페이지에서 경쟁 매물의 광고 문구를 **클릭 한 번에 CSV 파일로 자동 수집**하는 공인중개사 전용 크롬 확장 프로그램입니다.

**주요 기능:**

✅ 네이버 부동산에서 매물 클릭 한 번으로 동일 조건 경쟁 매물 자동 탐색  
✅ 각 경쟁 매물의 **한줄 특징·상세 설명** 일괄 수집  
✅ 수집 결과를 **CSV 파일**로 즉시 내보내기  
✅ 가격 오차 범위·면적 오차 범위 옵션으로 경쟁 매물 범위 조정 가능  
✅ 수집된 CSV를 **Gemini AI 도구**에 연결해 나만의 광고 문구 자동 생성  

### 사용 흐름

1. 크롬 웹스토어에서 네부스캐너 설치 (무료)
2. 네이버 부동산에서 등록할 매물 페이지 접속
3. 확장 프로그램 버튼 클릭 → 경쟁 매물 광고 문구 자동 수집
4. CSV 파일 저장 → Gemini AI 도구에 업로드
5. AI가 나만의 경쟁력 있는 광고 문구 생성

광고 문구 벤치마킹에 쓰던 시간을 본연의 중개 업무에 돌릴 수 있습니다.

## 도구 2. 전세낀 — 아파트 단지 내 전세낀·급매 매물 자동 전수조사

### 이런 분께 필요합니다

손님이 "이 단지에서 전세 끼고 살 수 있는 매물 있어요?"라고 물어볼 때, 지금은 어떻게 찾으시나요?

단지 내 매매 매물 수백 개를 일일이 열어보며 특징란에 '전세낀', '전세 승계' 등의 키워드가 있는지 직접 확인하고 계시다면, 이 도구가 그 시간을 완전히 없애줍니다.

### 전세낀이 하는 일

**전세낀**은 아파트 단지 내 **전체 매매 매물을 자동으로 전수조사**하고, 원하는 키워드(전세낀, 급매, 전세 승계 등)가 포함된 매물만 걸러내는 공인중개사 전용 크롬 확장 프로그램입니다.

**실제 성능:** 690개 매물을 30분 만에 전수조사 완료 (프로그램 실행 후 다른 업무 병행 가능)

**주요 기능:**

✅ 단지 내 전체 매매 매물 **자동 전수조사**  
✅ 원하는 키워드로 검색 — **전세낀, 급매, 전세 승계** 등 다양한 조건 적용 가능  
✅ 조사 완료 후 **키워드 재검색** 즉시 가능 (재수집 불필요)  
✅ 결과를 **CSV 파일**로 내보내 엑셀에서 추가 활용 가능  
✅ 매물별 **네이버 부동산 링크** 제공으로 담당 중개사 문의 간편  

### 사용 흐름

1. 크롬 웹스토어에서 전세낀 설치 (무료)
2. 네이버 부동산 아파트 단지 매매 매물 목록 페이지 접속
3. 확장 프로그램 실행 → 자동 전수조사 시작 (백그라운드 진행)
4. 완료 후 원하는 키워드로 즉시 필터링
5. 찾은 매물 링크로 바로 이동하거나 CSV로 저장

## 자주 묻는 질문 (FAQ)

**Q. 두 프로그램 모두 정말 무료인가요?**  
A. 네, 크롬 웹스토어에서 완전 무료로 설치할 수 있습니다. 별도 회원가입이나 결제 없이 바로 사용 가능합니다.

**Q. 컴퓨터 사용에 익숙하지 않아도 쓸 수 있나요?**  
A. 크롬 브라우저와 네이버 부동산 사용이 가능하다면 충분합니다. 크롬 웹스토어 설명과 각 도구 안내를 참고해 사용하시면 됩니다.

**Q. 네이버 부동산 계정이 필요한가요?**  
A. 네이버 부동산 일반 접속 환경에서 동작하므로, 별도의 특수 계정 없이 사용 가능합니다.

**Q. 전세낀으로 어떤 키워드든 검색할 수 있나요?**  
A. 자유 입력 방식이므로 '전세낀', '급매', '전세 승계', '즉시 입주' 등 원하는 키워드를 직접 입력하면 됩니다. 조사 완료 후 키워드를 바꿔서 재검색도 즉시 가능합니다.

**Q. 네부스캐너의 AI 문구 생성은 어떻게 작동하나요?**  
A. 네부스캐너로 수집한 CSV 파일을 Gemini AI 도구에 업로드하면, AI가 경쟁 매물의 문구를 분석해 나만의 광고 문구를 자동 생성해 줍니다. Gemini 계정(무료)이 필요합니다.

**Q. 사용 중 문제가 생기면 어떻게 하나요?**  
A. 각 도구의 카카오 오픈톡 채널에서 문의하시면 됩니다. 현직 중개사님들의 피드백을 바탕으로 지속 개선 중입니다.
