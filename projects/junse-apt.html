전세낀 급매물 찾기

크롬 웹스토어에서 설치

네이버 부동산 단지에서 매매 매물을 자동으로 전수 조사해 전세낀·급매 등 원하는 키워드만 골라냅니다.

단지 내 키워드 검색 전세 끼고 매수 등 특수 조건 크롬 확장 프로그램

주요 기능

  • 단지 내 전체 매매 매물 자동 전수 조사
  • 원하는 키워드로 검색 — 전세낀, 급매 등 다양한 조건 적용 가능
  • 조사 완료 후 키워드 재검색 즉시 가능 (재수집 불필요)
  • 결과를 CSV 파일로 내보내 엑셀에서 추가 활용 가능
  • 매물별 네이버 부동산 링크를 제공해 중개사 문의가 간편함

사용 방법

1단계

새로운 네이버 부동산에서 확장 프로그램으로 매물·단지 정보를 불러오는 절차입니다.

  1. 새로운 네이버 부동산(fin.land)에서 매물 선택
  2. ‘전세낀 급매물’ 확장 프로그램 클릭
  3. 매물 번호 자동 입력 후 ‘정보 확인’ 버튼 클릭
  4. 단지 정보를 확인하고 ‘매물 수집’ 클릭

2단계

옵션 설정하기

  1. 톱니바퀴 아이콘으로 옵션 창 열고 닫기
  2. 기보증금: 매물의 기보증금 여부 확인
  3. 키워드: 기본 키워드와 사용자 설정 키워드로 매물 검색

3단계

매물 수집이 완료되면 새 탭이 열립니다.

4단계

분석 진행 상황을 볼 수 있습니다.

  1. 총 수집된 매물 수와 분석을 완료한 매물 수를 확인할 수 있습니다.
  2. 검색된 매물: 조건에 맞는 매물 수입니다.
  3. 분석 예상 소요 시간이 표시됩니다.
  4. 분석 중에 다른 작업을 하셔도 됩니다.

주의: 분석이 진행 중일 때는 브라우저 상단의 ‘취소’ 버튼을 누르지 않습니다.

5단계

분석 완료

  1. 분석이 완료되면 알림이 옵니다.
  2. ‘분석 결과 보기’를 통해 리포트를 볼 수 있습니다.

6단계

분석 결과 보기

  1. 검색 결과를 CSV 형식으로 다운로드
  2. 버그 제보 및 문의
  3. 새로운 키워드로 재검색 가능
  4. 기보증금이 기재된 매물
  5. 포함된 키워드 개수를 볼 수 있습니다
  6. 매물 상세 보기가 펼쳐집니다
  7. 검색된 키워드는 오렌지색으로 표시됩니다

7단계

상세 보기

  • 가. 각 항목은 정렬 가능합니다
  • 나. 키워드는 오렌지색으로 보입니다
  • 다. 해당 매물로 이동할 수 있습니다
